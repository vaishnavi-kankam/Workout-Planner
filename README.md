# 🏋️ Workout Planner

A MERN Stack fitness tracking application.

## Features
- User Registration & Login
- Dashboard
- Add/Edit/Delete Workouts
- Workout History
- BMI Calculator
- Goals Tracker
- Water Tracker
- Sleep Tracker
- Diet Planner
- Workout Calendar
- Weekly Workout Plan
- Exercise Details

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Bootstrap
